# clinical-data-eda-python

# Análise de Dados Clínicos (Gerontologia)

Este projeto realiza o tratamento, limpeza e análise estatística descritiva de uma base de dados clínicos e sociodemográficos de idosos, focando nos índices de vulnerabilidade (IVCF) e estado cognitivo (MEEM).

## 🛠️ Tecnologias Utilizadas
* Python (Pandas)
* Google Planilhas (Tabelas Dinâmicas e Visualização)
